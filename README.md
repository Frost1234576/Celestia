# Celestia
A custom Tauri v2 IDE with a React front-end and native compatibility with Stella and Orion.

- [x] Initial Setup
- [ ] Flexible Layout
- [ ] Stella Linting & Highlighting
- [ ] YML Highlighting & Utils
- [ ] Multiple Concurrent Terminals
- [ ] Run/Build/Deploy Custom Workflows
